## Objective

A new user can submit the sign-up form with **valid** profile data and sees a confirmation message that matches **`expected_message`** on the row.

## Preconditions

- The sign-up screen is available.
- The email used for registration is not already registered (or use a unique address per your test plan).

## Test data

Authoritative field values are in **`testData/SignUp.xlsx`**, Sheet **Sheet1**, row **`tc_id`** = **TC004**. Do not duplicate cell values here; open the file when executing or reviewing the case.

## Steps

1. Open the **sign-up** / **register** page.
2. Enter **first name** and **last name** from the row.
3. Enter **email**, **phone**, and **password** using values that satisfy Preconditions (from the sheet where provided, or unique values if the sheet defers them).
4. Select **occupation** and **gender** (and complete any other mandatory fields) per the row.
5. Click **Register** / **Sign up** (or equivalent).
6. Confirm a success notification appears and its text matches **`expected_message`** on the row.

## Expected result

- The message shown after successful registration matches **`expected_message`** from the sheet (subject to application wording).
