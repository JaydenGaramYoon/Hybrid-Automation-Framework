## Objective

Submitting login with an **invalid email** (per the row) shows an error message that matches **`expected_message`**.

## Preconditions

- The login screen is reachable.

## Test data

Authoritative field values are in **`testData/LogIn.xlsx`**, Sheet **Sheet1**, row **`tc_id`** = **TC002**. Use **`invalid_email`** and related columns as defined there.

## Steps

1. Go to the **login** screen.
2. In **email**, enter the value from **`invalid_email`** on the row (invalid format for this case; **`focus`** on the row should indicate email is under test).
3. In **password**, enter the value from **`password`** on the row.
4. Click **Sign in** / **Login**.
5. Confirm the user is **not** logged in and an error is shown; the error text must match **`expected_message`** on the row.

## Expected result

- The message shown matches **`expected_message`** from the sheet (subject to application wording).
