# TC001 - As a registered user, I want to sign in with valid email and password so that I can access my account.

## Objective

A user who submits **valid** email and password sees a success feedback message that matches **`expected_message`** on the data row.

## Preconditions

- The application is available for testing.
- A user account exists for the email and password defined on the row (or equivalent demo data).

## Test data

Authoritative field values are in **`testData/LogIn.xlsx`**, Sheet **Sheet1**, row **`tc_id`** = **TC001**. Do not duplicate cell values here; open the file when executing or reviewing the case.

## Steps

1. Open the application and go to the **login** screen.
2. In the **email** field, type the value from column **`email`** on the row.
3. In the **password** field, type the value from column **`password`** on the row.
4. Activate **Sign in** / **Login** (or the primary submit control for the form).
5. After the response appears, read the success feedback (toast, banner, or inline text) and check that its full text matches **`expected_message`** on the row.

## Expected result

- The message shown after a successful login matches **`expected_message`** from the sheet (subject to application wording).
